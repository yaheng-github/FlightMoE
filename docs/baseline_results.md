# FlightMoE 基线复现实验报告

## 1. 物理一致性邻接矩阵

### 方法
- 分阶段：6 种飞行阶段（hover/waypoint/velocity/circling/acce/dece）
- 状态离散化：mean/diff 分别做 L/H（10%/90% 分位数）
- 关联规则：NumPy 向量化 2-项集计数，对称置信度
- 伪恒定字段隔离：battery_temperature, GPS 恒定项

### 交付物
- `data/adjacency/adjacency_{phase}.npy` — 6 个 41×41 矩阵
- `data/adjacency/rules_{phase}.json` — 各阶段强规则列表
- 热力图（Top-K 稀疏化显示）

## 2. MAD-GAN 基线（Burst Expert）

### 实验设置
| 参数 | 值 |
|------|-----|
| 网络 | LSTM Generator + LSTM Discriminator |
| hidden_dim | 512 |
| latent_dim | 32 |
| batch_size | 32 |
| lr | 1e-4 |
| epochs | 50 |
| 训练数据 | train.npz 正常样本（5300 窗口）|
| 输入 | [batch, 128, 41] 时序窗口 |

### 调参过程
初始默认 `res_weight=0.2`（重构误差占 20%，判别器占 80%），结果不佳：

| 配置 | 闭集 AUC | 开集 AUC | 问题 |
|------|---------|---------|------|
| res=0.2 | 0.6541 | 0.6234 | 开集几乎全报异常 |

**关键发现**：通过子集快速搜索发现 res_weight 越大越好，重构误差才是主要信号。

### 最终最优结果（res_weight=0.9）

| 指标 | 闭集 (test_closed) | 开集 (test_open) |
|------|-------------------|------------------|
| **AUC-ROC** | **0.9168** | **0.8568** |
| F1 | 0.8787 | 0.8337 |
| Precision | 0.9257 | 0.8431 |
| Recall | 0.8362 | 0.8245 |
| Best Threshold | 0.7960 | 0.7231 |

### 结论
- 仅调整 `res_weight` 从 0.2 到 0.9，AUC 提升 +0.26（闭集）/ +0.23（开集）
- 重构误差是 MAD-GAN 的主要异常信号，判别器分数权重过高会干扰检测
- MAD-GAN 对明显偏离正常分布的突变异常检测效果好

- 像素值归一化到 [-1, 1]

| Precision | 0.7687 | 0.5678 |
| Recall | 0.8190 | 0.8489 |
| Best Threshold | 0.4958 | 0.4973 |

### 训练记录
- 训练 50 epochs 完成
- 中间验证最佳 AUC: 0.8992（epoch 16）
- 最终模型在测试集上的性能如上表

### 结论
- GANomaly 对闭集异常（与训练集同分布）检测效果较好，AUC 0.85

## 4. HMM Drift Expert（Drift Expert）

### 实验设置
| 参数 | 值 |
|------|-----|
| 网络 | GaussianHMM |
| 输入 | 窗口级统计特征 [N, 123]（均值/标准差/极差）|
| PCA 维度 | 8 |
| 协方差类型 | diag |
| BIC 状态数范围 | 2 ~ 12 |
| 异常分数 | -log-likelihood（越高越异常）|

### 关键改进
原论文方法（连续时间步级 HMM + Hellinger 距离）在 RflyMAD 窗口化数据上完全失效（AUC ≈ 0.5），原因：
1. 窗口边界不连续，HMM 学到错误的转移结构
2. 41 维传感器经 PCA 后，小窗口（100 步）的 Hellinger 协方差估计不稳定
3. 测试集正常数据包含训练集未覆盖的飞行阶段，分布偏移被误判为异常

**改进方案**：提取窗口级统计特征（均值/标准差/极差），让 HMM 学习"正常窗口的特征模式"，用 log-likelihood 作为异常分数。这样每个窗口是一个独立样本，不依赖时间连续性，对分布偏移更鲁棒。

### 最终评估结果

| 指标 | 闭集 (test_closed) | 开集 (test_open) |
|------|-------------------|------------------|
| **AUC-ROC** | **0.7819** | **0.7569** |
| F1 | 0.8098 | 0.7804 |
| Precision | 0.6832 | 0.6425 |
| Recall | 0.9938 | 0.9935 |
| Best Threshold | 17.9160 | 18.1662 |

### 结论
- 窗口级统计特征 + HMM log-likelihood 有效适配了 RflyMAD 数据
- 闭集/开集 AUC 均达到 0.75+，Recall 接近 1.0（几乎不漏检）
- 开集 Precision 偏低（0.64），说明对分布偏移敏感，但总体可用

## 5. GDN 基线（Consistency Expert）

### 实验设置
| 参数 | 值 |
|------|-----|
| 网络 | GDN（Graph Attention Network）|
| 输入 | 窗口内滑动 `[41, slide_win]` 预测下一步 `[41]` |
| slide_win | 32 |
| slide_stride | 5（训练）/ 1（测试）|
| dim | 64 |
| topk | 5 |
| batch_size | 128 |
| lr | 0.001 |
| epochs | 100（早停 15）|

### 关键适配
- **分 6 阶段训练**：每个飞行阶段独立训练一个 GDN，各用各的物理一致性邻接矩阵
- **图结构替换**：GDN 默认全连接图 → 替换为物理一致性邻接矩阵（6 个阶段 6 张图）
- **PyG 2.x 兼容**：原始 GDN 针对 PyG 1.5，重写 `GraphLayer` 适配 PyG 2.x API
- **测试路由**：按 `phase_label` 路由到对应阶段的 GDN 模型

### 最终评估结果

| 指标 | 闭集 (test_closed) | 开集 (test_open) |
|------|-------------------|------------------|
| **AUC-ROC** | **0.9642** | **0.9122** |
| F1 | 0.9377 | 0.8826 |
| Precision | 0.9393 | 0.8691 |
| Recall | 0.9360 | 0.8965 |

### 各阶段闭集表现
| 阶段 | AUC | F1 | Precision | Recall |
|------|-----|----|-----------|--------|
| hover | 0.9359 | 0.9310 | 0.9669 | 0.8977 |
| waypoint | 0.9858 | 0.9416 | 0.9701 | 0.9147 |
| velocity | 0.9800 | 0.9516 | 0.9256 | 0.9791 |
| circling | 0.9798 | 0.9630 | 0.9527 | 0.9736 |
| acce | 0.9600 | 0.9768 | 0.9798 | 0.9738 |
| dece | 0.9499 | 0.9325 | 0.9535 | 0.9124 |

### 结论
- **分阶段 + 物理邻接矩阵**是 GDN 适配的关键：混训+全局平均图效果会差很多
- GDN 是目前四个 Expert 中效果最好的（闭集 AUC 0.96，开集 AUC 0.91）
- 图网络对传感器间**一致性破坏**（关联断裂）非常敏感，正好对应 Consistency Expert 的定位

## 6. 四大专家模型综合对比

### 6.1 核心指标对比

| Expert | 模块 | 检测目标 | 闭集 AUC | 闭集 F1 | 开集 AUC | 开集 F1 |
|--------|------|---------|---------|--------|---------|--------|
| **Burst Expert** | MAD-GAN | 时序突变 | 0.9168 | 0.8787 | 0.8568 | 0.8337 |
| **Spectral Expert** | GANomaly | 频域异常 | 0.8544 | 0.7930 | 0.7004 | 0.6805 |
| **Drift Expert** | HMM | 统计漂移 | 0.7819 | 0.8098 | 0.7569 | 0.7804 |
| **Consistency Expert** | GDN | 传感器一致性断裂 | **0.9642** | **0.9377** | **0.9122** | **0.8826** |

### 6.2 各模型优势与局限

| 模型 | 优势 | 局限 |
|------|------|------|
| **MAD-GAN** | 对突发突变敏感，闭集效果好 | 需调 `res_weight`，开集略降 |
| **GANomaly** | 频域特征对结构性异常有效 | 开集性能崩（AUC 0.70），分布偏移敏感 |
| **HMM** | 计算快，对缓慢漂移有理论基础 | 需改窗口级特征，原始方法失效 |
| **GDN** | **全局最优**，图结构显式建模传感器关联 | 需分阶段训练，依赖物理邻接矩阵 |

### 6.3 互补性分析

- **MAD-GAN** 强在**突发异常**（电机堵转、控制突变）
- **GANomaly** 强在**频域结构性异常**（振动、震荡）
- **HMM** 强在**统计分布漂移**（传感器 gradual degradation）
- **GDN** 强在**关联断裂**（油门加但电机不转、GPS 漂移但姿态正常）

→ **MoE Router 的价值**：不同异常类型激活不同 Expert，融合后应优于单一模型。
